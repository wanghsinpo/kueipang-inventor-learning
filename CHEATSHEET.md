## 🟥 已知坑（不要踩）

1. `Face.Geometry.Normal` 方向不可靠 → 用 Z 位置 + Area
2. SketchLines.AddByTwoPoints 必須用 SketchPoint（不是 Point2d）
3. `Edge.GeometryType=5124` 是圓邊（ObjectTypeEnum）
4. PowerShell `,@()` 內含表達式 parse 爆 → 拆出 scalar
5. PowerShell `$pid` reserved → 改 `$invPid`
6. Fillet 必須在 Chamfer 之前（順序錯會 break）
7. Inventor 處理 v3+ 後 RPC 失敗常見 → kill + restart
8. ChamferFeatures batch 失敗時用 per-edge fallback
9. ThreadFeatures.CreateThreadInfo 在 2027 API 不同
10. STEP/STL export 用 `Document.SaveAs(path, true)` 簡單，不用 translator
