## 🟥 Inventor COM 致命坑（已踩過）

1. **`Face.Geometry.Normal` 方向不可靠** — 用 `Z position + Area` 找 top/bottom face
2. **`SketchLines.AddByTwoPoints` 必須用 SketchPoint** — 不是 Point2d，否則 lines 不分享端點，Profile.AddForSolid 失敗
3. **`Edge.GeometryType = 5124`** 才是圓邊（ObjectTypeEnum）— 不是 38914 (CurveTypeEnum)
4. **`ChamferFeatures` 要 `AddUsingDistance`** — 不是 `CreateChamferDefinition`
5. **`Fillet 必須在 Chamfer 之前`** — fillet 後的 edge 變 spline 不能 chamfer
6. **Configurator 360 add-in** 在 Inventor 2027 PowerShell COM 啟動時會彈 BinaryFormatter exception — 卡住 GetTemplateFile 等待
7. **`ApplicationAddIns.ItemById` 方法 .Translator 屬性 = null** in 2027 — 用 `$doc.SaveAs(stepPath, $true)` 簡單方法 export STEP/STL
8. **`HoleFeatures.CreatePointPlacementDefinition`** 在 PowerShell COM 拋 ArgumentException — 暫時用 sketch + extrude cut 代替
9. **`ThreadFeatures.CreateThreadInfo`** 在 2027 不存在 — 用螺紋 pilot 孔（Ø3.3 for M4）暫代

## 🟦 PowerShell 已知坑

1. `,@()` 內含表達式 parser 爆 — 拆出 scalar
2. `$pid` reserved — 改名 `$invPid`
3. 內聯 `if(){}else{}` 在參數位置不能用 — 先賦值
4. PowerShell 5.1 不支援 `??` null-coalescing — 用 `if (-not ...)` 替代
5. `Join-Path` 三個 path components — 用 `[System.IO.Path]::Combine()` 替代
6. Here-string with `'` (apostrophe) 給 git commit -m 會 break — 用 `@'...'@` literal block

## 🟢 高效 Workflow

1. **Photo→CAD 建模**：
   - 先描述照片內容，不要套模板
   - 列每個 feature 對應照片哪部分
   - Feature 順序：rough → fillet → cut/hole → chamfer → cosmetic
   - 驗證對稱性（M4 是 0/90/180/270 還是 45° 對角？）

2. **Pipeline 維運**：
   - `refresh_all.ps1` 一次更新所有 dashboards
   - `validate_pipeline.ps1` 檢查所有 PASS 完整性
   - `quality_check.ps1` 找 borderline + back-calc 大 parts

3. **修 FAIL 部件**：
   - 看 BBox + cylinder count → 選 strategy
   - back-calc 永遠 vol PASS（但形狀可能差）
   - 複雜形狀 → 手動寫 manual_*.ps1
