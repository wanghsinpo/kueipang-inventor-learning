## 🔄 可以改進但不急的

### 1. Visual accuracy（視覺準確度）
- 早期 ~81 個 part 用 back-calc 救起 PASS，volume 對得上但形狀可能跟真檔差很多
- 改善方法：人工開啟真檔，看實際幾何（stepped shaft / multi-feature / etc.），寫對應的 manual fix
- 建議：每天挑 1-2 個來修，保留 visual diff 小於 5%

### 2. R1114 / R1115 真實幾何
- 目前用薄壁空心圓柱（OD104.9 × ID101.1 × wall1.9mm × H119.7）vol 0% 但形狀差很多
- 真檔是 stepped shaft + 自由曲面（11 個 BSpline/torus 面）
- 改善方法：用卡尺實際量幾個關鍵 dimensions，寫多 step 模型

### 3. 8 個 SKIP 部件
- R355, R370-R374, R460, R478, R748 — 都因為 .ipt 開不了無法處理
- 改善方法：用 Inventor 直接 repair / re-save 這些檔案，或要求重新 export 真檔

### 4. motor_flange 視覺更接近照片
- 目前 v19 是「合理近似」但不是 1:1 對應
- 改善方法：用 caliper 量幾個關鍵尺寸，更新 param 值

### 5. ThreadFeatures cosmetic threads
- 目前 4 個 M4 螺紋孔只是 Ø3.3 通孔（沒有 thread display）
- 改善方法：Inventor 2027 API 的 ThreadFeatures.Add 需要 ThreadInfo 物件，
  CreateThreadInfo 方法在 PowerShell 中無法直接呼叫 — 需找替代 API path

### 6. Inventor HoleFeature API
- 目前 holes 用 sketch + extrude cut（功能上正確但語意上不是 Hole feature）
- 改善方法：CreatePointPlacementDefinition 在 PS 中 fail（Value out of range），
  需用 Inventor add-in 或 .NET 直接呼叫
