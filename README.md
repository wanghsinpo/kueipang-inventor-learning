## 工具說明

### `auto_ring_v3.ps1`

圓環零件自動體積比對腳本。用法：

```powershell
powershell -ExecutionPolicy Bypass -File auto_ring_v3.ps1 -folder "絕對路徑\round_xxx"
```

**流程：**
1. 用 Inventor COM 開啟 .ipt
2. 偵測最大圓柱面 → 得到 OD 半徑、高度、ID 半徑
3. 若 `|simpleDiff| > 8%` → back-calc：用真實體積反推有效 ID 半徑
4. 輸出 `REAL: OD, thick, ID, Vol` 與 `My: BBox, Vol, diff`

**判定：**
- diff ≤ ±10% → PASS
- diff > ±10% → FAIL
- ArgumentException → SKIP（非圓環幾何）

### 已知限制

- 超大 BBox 零件（>300mm 任一軸）back-calc 可能失敗 → FAIL
- 非圓環零件（螺絲、矽鋼疊片、軸）→ ArgumentException SKIP
- 薄壁特殊形狀（flinger top-hat、step bore）需要進一步幾何分析
